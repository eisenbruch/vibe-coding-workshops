# Are You a Top or a Bottom?

A parody quiz styled after early 2000s magazine flowcharts. Asks 10 randomized lifestyle questions (snacks, vacation activities, emojis, etc.) then delivers a completely random result.

## Features

- Y2K magazine aesthetic with hot pink gradients and playful typography
- Slide transitions between questions
- Shuffled question order each playthrough
- Random outcome regardless of answers (it's a joke quiz)

## Usage

Open `index.html` in a browser. No build step required.
