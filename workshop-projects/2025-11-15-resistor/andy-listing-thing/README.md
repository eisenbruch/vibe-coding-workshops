# Band Tier Console

A drag-and-drop ranking interface. Built with React.

## Features

- Drag items from the available list to the ranking stack
- Reorder ranked items by dragging within the stack
- Neon glow effects and scanline overlay
- Responsive two-panel layout

## Usage

Open `index.html` in a browser. Uses CDN-hosted React 17 and Babel for in-browser JSX transformation.
