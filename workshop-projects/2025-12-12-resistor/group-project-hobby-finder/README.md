# Ye Olde Hobby Finder

An interactive hobby discovery tool with an Old English medieval manuscript theme. Swipe through pursuits, duel your favorites, and discover what truly stirs thy spirit.

## Features

### Peruse (Discover)
Tinder-style card swiping through 30 curated hobbies. Each card displays:
- Hobby name and emoji
- Category (Creative, Nature, Fitness, etc.)
- Description
- Tags (Solo, Outdoors, Relaxing, etc.)

**Controls:**
- Drag cards left/right
- Click buttons: ✕ (Nay), ★ (Super), ♥ (Yea)
- Keyboard: ← (pass), → (like), ↑ (super like)

### Duel (Decide)
Head-to-head matchups between two hobbies. Pick the one that calls to you more. When a hobby wins 3 duels, it's automatically added to your collection.

### Thy Collection (Saved)
View and manage your saved hobbies. Remove any that no longer interest you with the "Banish" button.

## Hobbies Included

30 activities selected for variety and accessibility:

- **Low-impact fitness:** Swimming, Cycling, Yoga, Hiking
- **Creative:** Photography, Writing, Drawing, Music Production, Pottery
- **Nature:** Gardening, Birdwatching, Fishing, Astronomy
- **Crafts:** Woodworking, Leatherworking, Metalworking
- **Social:** Board Games, Dancing, Volunteering
- **Strategic:** Chess
- **Culinary:** Cooking, Brewing
- **Adventure:** Rock Climbing, Kayaking, Disc Golf, Archery
- **Wellness:** Meditation
- **Educational:** Learning Languages
- **Tech/Creative:** Podcasting

## Design

Medieval manuscript aesthetic featuring:
- Parchment textures and paper grain overlay
- **UnifrakturMaguntia** blackletter font for headers
- **IM Fell English** serif for body text
- **Cinzel** Roman inscriptional style for buttons
- Sepia/brown palette with gold, crimson, and forest green accents
- Ornate borders and decorative flourishes

## Usage

Open `hobby-finder.html` in any modern browser. No build step or dependencies required.

## Browser Support

Works in all modern browsers (Chrome, Firefox, Safari, Edge).
